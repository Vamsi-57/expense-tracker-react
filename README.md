# Expense Tracker (React)

This is a React version of the [Vamsi-57 JS Expense Tracker](https://github.com/Vamsi-57/Vamsi-57projects/tree/master/expense-tracker). It uses functional components with hooks and the context API

## Usage
```
npm install

# Run on http://localhost:3000
npm start

# Build for prod
npm run build
```
